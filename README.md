# web-clj

Toy project to showcase Clojure for the web.

## Prerequisites

You will need [Leiningen][] 2.0.0 or above installed.

[leiningen]: https://github.com/technomancy/leiningen

## Running

To start the server, run:

    lein run

Starts the Aleph HTTP server at `:8080`.
